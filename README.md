\# 🏦 Bluestock Fintech — Mutual Fund Analytics Capstone



\## Project Overview

A full-stack Mutual Fund Analytics Platform built as a 7-day

capstone project for Bluestock Fintech Pvt. Ltd.



\## Tech Stack

\- Python 3.12 | Pandas | NumPy | Matplotlib | Seaborn | Plotly

\- SQLite | SQLAlchemy | Jupyter Notebook | Power BI



\## Project Structure

bluestock-fintech-capstone/

├── data/

│   ├── raw/            ← Original CSV + API fetched files

│   └── processed/      ← Cleaned and transformed data

├── notebooks/          ← Jupyter analysis notebooks

├── scripts/            ← Python ETL scripts

├── sql/                ← SQL schema and queries

├── dashboard/          ← Power BI dashboard file

├── reports/            ← Final report and presentations

├── requirements.txt    ← All Python dependencies

└── README.md           ← You are here



\## Setup Instructions

1\. Clone the repo

2\. Create virtual environment: python -m venv venv

3\. Activate: venv\\Scripts\\activate

4\. Install dependencies: pip install -r requirements.txt

5\. Launch Jupyter: jupyter notebook



\## Data Source

AMFI India (amfiindia.com) | mfapi.in | NSE/BSE Public Data
