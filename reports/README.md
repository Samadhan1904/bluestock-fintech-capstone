# reports/

- `Final_Report.pdf` - 15-20 page final project report covering
  Executive Summary, Data Sources, ETL Pipeline, EDA Findings,
  Performance Analysis, Dashboard Screenshots, Recommendations
  and Limitations.

- `Bluestock_MF_Presentation.pptx` - 12-slide presentation deck.

- `charts/` - All exported chart PNGs/JPEGs from Days 3-6 plus
  Power BI dashboard page screenshots.

- `data_quality_day1.txt` - Day 1 data quality summary.
- `data_dictionary.md` - Column-level documentation for all DB tables.
- `query_results_day2.txt` - Output of the 10 SQL queries.

## Regenerating

```
python scripts/generate_report.py
python scripts/generate_presentation.py
```
