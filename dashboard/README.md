# dashboard/

- `bluestock_mf_dashboard.pbix` - Power BI dashboard with 4 pages:
  Industry Overview, Fund Performance, Investor Analytics, and
  SIP & Market Trends. Every page has 2+ interactive slicers.

- `Dashboard.pdf` - Static PDF export of all 4 pages.

## Opening

Open `bluestock_mf_dashboard.pbix` in Power BI Desktop. Data sources
point to CSVs in `data/processed/` - use "Refresh" if those files
have been regenerated.
